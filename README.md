# TZ
ТЗ haulmont
Банковское приложение, позволяющее оформить кредит на клиента и составить для него аннуитетный график платежей.

Требования
-------------
Java Development Kit (JDK) 11

Maven 3.8.1

Сборка и запуск
-------------
	mvn clean compile
	mvn spring-boot:run
